# Task Management API - Project Guide

## GTD Workflow

This project follows the GTD (Get Tasks Done) workflow. All work is organized into phases, each with a plan and verification.

### Key Commands

- `/gtd-plan-phase [N]` — Plan a phase
- `/gtd-discuss-phase [N]` — Discuss phase approach
- `/gtd-progress` — Check progress
- `/gtd-verify-work` — Verify phase deliverables

## Project Context

**Core Value:** A clean, reliable task management API with minimal feature set — create, track, and assign tasks with persistent state.

**Stack:** Node.js + Express.js + Prisma + PostgreSQL + JWT
