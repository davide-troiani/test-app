**GTD — top commands**

```text
/gtd-new-project           Initialize a project (greenfield)
/gtd-map-codebase          Map an existing codebase (brownfield)
/gtd-plan-phase <N>        Create a phase plan
/gtd-export-phase-issues   Export planned tasks to GitHub issues
/gtd-work-task-issue       Work one exported task; also completes phases
/gtd-orchestrate-tasks     Coordinate a small exported task batch
/gtd-progress              Where am I, what's next
/gtd-quick                 Small ad-hoc task with GTD guarantees
/gtd-fast "<task>"         Trivial inline task — no subagents
/gtd-debug "<symptom>"     Persistent debug session (survives /clear)
/gtd-capture               Save an idea / todo / note
/gtd-work-task-issue --phase <N>      Work the next exported task issue
```

More: `/gtd-help` (default tour) · `/gtd-help --full` (everything) · `/gtd-help <topic>` (one section)
